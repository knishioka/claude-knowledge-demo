## Summary（3行）
- 1セッション視聴時間＝1回の訪問で動画を視聴した平均時間
- 現状値は12分、目標は15分（+3分）
- 継続率の補助指標として「エンゲージメントの深さ」を測定

## Body

**定義**:
```
1セッション視聴時間 = 総視聴時間 / 総セッション数
```

**計測ルール**:
- セッション＝アプリ起動から終了まで（30分無操作でタイムアウト）
- 視聴時間＝動画再生中の時間のみ（一時停止中は含まず）
- バックグラウンド再生は含む

**現状値の内訳**:
| ユーザータイプ | 平均視聴時間 |
|---------------|-------------|
| 新規（7日以内） | 8分 |
| 継続（7-30日） | 14分 |
| ロイヤル（30日超） | 22分 |
| **全体平均** | **12分** |

**目標+3分の根拠**:
新規ユーザーの視聴時間を8分→12分に改善できれば、全体平均は15分程度になる試算。
