# /capture コマンド

素早くinboxにメモを追加するためのテンプレートを生成します。「今すぐDistillしない」前提で、後でdistillしやすい形に整えます。

## 入力
- `$ARGUMENTS`: メモのタイトルまたは文脈（例: `新しいアイデア`, `MTGメモ`, `気づき`）

## 処理手順

### Step 1: メモの種類を判定する

入力から、以下のメモタイプを推定してください:

| タイプ | キーワード例 | テンプレート |
|--------|-------------|-------------|
| **meeting** | MTG, 会議, ミーティング | 議事録テンプレート |
| **idea** | アイデア, 思いついた, 提案 | アイデアメモテンプレート |
| **insight** | 気づき, 発見, 学び | インサイトテンプレート |
| **task** | タスク, TODO, やること | タスクメモテンプレート |
| **question** | 疑問, 質問, 確認 | 質問メモテンプレート |
| **general** | その他 | 汎用テンプレート |

### Step 2: テンプレートを生成する

#### 議事録テンプレート（meeting）
```markdown
# {タイトル}

**日時**: YYYY-MM-DD HH:MM
**参加者**:
**目的**:

---

## アジェンダ
1.
2.
3.

---

## 議事内容

### 1. {議題}
-

---

## 決定事項
- [ ]

## 未決事項
-

## リスク
-

## 次アクション
| アクション | 担当 | 期限 |
|-----------|------|------|
| | | |

---
*次回: YYYY-MM-DD*
```

#### アイデアメモテンプレート（idea）
```markdown
# {タイトル}

**日時**: YYYY-MM-DD
**文脈**: {どんな状況で思いついたか}

---

## アイデアの概要
（3行以内で）

## 背景・きっかけ
-

## 期待される効果
-

## 懸念・リスク
-

## 次のステップ
- [ ] {まず何をすべきか}

---
*関連: [[...]]*
```

#### インサイトテンプレート（insight）
```markdown
# {タイトル}

**日時**: YYYY-MM-DD
**情報源**: {どこで知ったか}

---

## 気づき（1行）


## なぜ重要か
-

## どう活かすか
-

## 次の質問
-

---
*関連: [[...]]*
```

#### 質問メモテンプレート（question）
```markdown
# {タイトル}

**日時**: YYYY-MM-DD
**緊急度**: 高 / 中 / 低

---

## 質問
（明確に1文で）

## 背景
-

## 想定される回答
1.
2.

## 確認先
-

## 期限
-

---
*回答が得られたら [[note-...]] に記録*
```

#### 汎用テンプレート（general）
```markdown
# {タイトル}

**日時**: YYYY-MM-DD

---

## 概要
-

## 詳細
-

## 次のアクション
- [ ]

---
*Distill時の分類候補: #decision / #risk / #data / #insight*
```

### Step 3: ファイル名を生成する

形式: `YYYY-MM-DD_{type}_{short-title}.md`

例:
- `2025-12-26_meeting_weekly-sync.md`
- `2025-12-26_idea_new-feature.md`
- `2025-12-26_insight_user-behavior.md`

### Step 4: 出力

```markdown
## /capture 結果

### テンプレート
{生成したテンプレート}

### 保存先
`knowledge/inbox/YYYY-MM-DD_{type}_{title}.md`

---
このテンプレートをinboxに作成してよいですか？
（作成後、内容を入力してください）
```

## 制約
- **30秒で入力できる量に留める**: 詳細は後でDistillする
- **構造化を優先**: 後でDistillしやすいセクション分け
- **Distill候補タグを残す**: `#decision / #risk / #data` などのヒント

## ショートカット

```
/capture MTG    → 議事録テンプレート
/capture idea   → アイデアメモテンプレート
/capture ?      → 質問メモテンプレート
```
