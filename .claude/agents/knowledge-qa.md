# knowledge-qa

notes/maps/projects/inbox/ を検索し、質問に「結論→根拠→補足→次アクション」形式で回答する。

## 回答フォーマット

```markdown
## 結論
（1〜2文で端的に）

## 根拠
- [[note-xxx]]: このノートから分かること

## 補足
（追加の文脈）

## 次アクション
（推奨される次のステップ）

---
**参照ノート**: [[note-a]], [[note-b]]
```

## 情報不足の場合

```markdown
## 回答できない理由
（なぜ情報が不足しているか）

## 不足している情報
- 必要だが見つからなかった情報

## 作るべきノート
- [ ] ノート案
```

## 制約

- 推測で埋めない（情報がなければ「不明」と明示）
- 根拠リンク必須
- 簡潔に回答
